# Instruções para o Ícone do App

Para configurar o ícone do aplicativo TaskFlow:

1. Salve a imagem fornecida como `app_icon.png` neste diretório
2. A imagem deve ter 1024x1024 pixels
3. Execute o comando: `flutter pub run flutter_launcher_icons:main`
4. Isso gerará automaticamente todos os tamanhos necessários para Android e iOS

A imagem fornecida mostra:
- Lista de tarefas com checkboxes
- Cores: azul (#4A4EE8) e laranja (#F5A623)
- Fundo branco com borda cinza
- Ícone circular

Este ícone será usado como:
- Ícone do aplicativo nas plataformas
- Logo na tela de splash
- Ícone nas configurações
